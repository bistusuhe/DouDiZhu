# 斗地主规则模拟对局

一个基于原生 HTML/CSS/JavaScript 的交互式斗地主规则演示页面。

本项目通过“上家出牌 + 你方应对”的模拟流程，帮助快速理解常见牌型、压牌规则、叫地主流程与基础策略。

## 项目特点

- 纯前端实现，无第三方依赖
- 开局规则引导遮罩，包含地主与农民胜利条件和策略说明
- 开始体验后可选择“叫地主 / 不叫地主”
- 左侧展示 3 张地主底牌（未叫地主前为倒置牌）
- 叫地主后底牌翻开并加入手牌，自动排序
- 支持手牌选择、出牌、过牌、提示
- 牌型合法性校验与压制关系校验
- 无可压时弹窗提示，可再来一局或结束体验
- 牌型介绍区支持翻面查看说明
- 舞台尺寸与缩放适配（桌面 / 紧凑模式）

## 当前支持的牌型

- 单张
- 对子
- 三张
- 三带一
- 三带二
- 顺子
- 连对
- 飞机
- 飞机带单
- 飞机带对
- 炸弹
- 王炸



## 交互流程

1. 进入页面后显示开局规则引导层
2. 点击“开始体验”进入“是否叫地主”选择
3. 选择“叫地主”：底牌翻开并并入手牌
4. 选择“不叫地主”：直接开始对局
5. 轮到你时可使用“出牌 / 过牌 / 提示”
6. 当无法压住上家牌型时出现弹窗

## 项目结构

- `index.html`：页面结构（牌桌、引导层、弹窗、牌型区）
- `styles.css`：界面样式（牌桌、卡牌、动效、响应式）
- `script.js`：对局逻辑（牌型识别、压制规则、提示、流程控制）

## 关键可配置项

可在 `script.js` 中按需调整：

- `USER_HAND_SEED`：初始手牌
- `LANDLORD_SEED`：地主底牌
- `generateOpponentPlays()`：对手出牌脚本（当前飞机类型每局只出一次）

可在 `styles.css` 中按需调整：

- `:root` 下的 `--stage-w`、`--stage-h`、`--card-w`、`--card-h`
- `.stage-shell::before`：木质边框样式
- `.start-overlay`：开局遮罩层样式

## 说明

本项目用于规则学习与交互演示，不追求完整联网对战逻辑。
